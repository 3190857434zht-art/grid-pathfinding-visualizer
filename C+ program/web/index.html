操作说明
  • 编辑：先选工具，再左键编辑；右键任意位置可快速擦除。
  • 运行寻路：选算法后点主按钮，或按 Enter。
  • 随机散点障碍：在当前地图上随机添加障碍，不重置地图。
  • 算法对比：勾选算法对比后，点击下方各算法即可播放寻路动画。
  • 填充：左键或右键将格子变为空地，可快速修正误画的障碍。
  • 快捷键：1障碍 2填充 3起点 4终点 · Enter 运行 · Esc 停动画
